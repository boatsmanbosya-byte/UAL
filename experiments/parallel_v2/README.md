# Параллельная реализация (v2.0) — экспериментальная, НЕ часть пакета ual/

Альтернативная реализация из параллельной переписки. Полезные идеи
(персистентность L_t между запусками) перенесены в основной пакет
отдельно, без повторения найденной здесь ошибки.

ИЗВЕСТНЫЙ БАГ: save/load не работает после настоящей перезагрузки
процесса — Symbol как подкласс str обходит кастомный JSON-энкодер,
теряя различие "символ vs литерал" в leval. Подробности и воспроизведение
ошибки — см. ual_journal.md, раздел 21.
